Int and Float use spinbox

Set Params
  Set: <select>
    Options:
      "Mandelbrot Set"
      "Julia Set (Not Implemented)"
      "Burning Ship Fractal (Not Implemented)"
      "Custom Polynomial (Not Implemented)"
  For "Mandelbrot Set":
    Exponent: <int>
    Zx: <float>
    Zy: <float>
Location Params
  X: <float>; spinbox speed based on zoom
  Y: <float>; spinbox speed based on zoom
  Zoom: <float>; spinbox scaling logarythmic
  Log Zoom: <checkbox>
  Rotation: <float>
  SkewX: <float>
  SkewY: <float>
Render Params
  Render Mode External: <select>
    Options:
      "Iteration Count"
      "Exterior Distance Estimate"
  For "Iteration Count":
    Iteration Count: <int>; range: x >= 0
    Iteration Count Scaling: <float>
    Normalized: <checkbox>
  Render Mode Internal: <select>
    Options:
      "Loop Iteration Count"
      "Interior Distance Estimate"
  For "Loop Iteration Count":
    Loop Iteration Count: <int>; range: x >= 0
    Loop Iteration Count Scaling: <float>
  Log View: <checkbox>
  Precision: <select>
    Options:
      "Float"
      "Double"
      "Long Double" (if not available on processor then will say "Long Double (Emulated)" emulated using GMP or MPFR or MPIR)
      "Quad (GMP)" (possibly MPFR or MPIR instead)
      "Oct (GMP)" (possibly MPFR or MPIR instead)
      "BigDecimal (GMP)" (possibly MPIR instead)
Calculation Params
  Calculation: <select>
    Options:
      "Normal"
      "Optimized Algo"
      "Vector Operations"
      "Shader"
  Threads: <int>; range: x >= 0
  Edge Tracing: <checkbox>
  Blitting: <select>
    Options:
      "FillRect (very slow)"
      "Bitmap + SetPixel (slow)"
      "Bitmap + PixelArray (medium)"
      "Vulkan (fast)"
  Render to File: <button>
UI Params
  Dragging Mode: <select>
    Options:
      "Normal Mode"
      "Centered Cursor"
Misc:
  Version: <info>
  Check for Updates: <button>
