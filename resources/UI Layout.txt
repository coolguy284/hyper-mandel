Int and Float use spinbox

Set Params
	Set: <select>
		Options:
			"Mandelbrot Set" (default)
			"Julia Set (Not Implemented)"
			"Burning Ship Fractal (Not Implemented)"
			"Custom Polynomial (Not Implemented)"
	For "Mandelbrot Set":
		ExponentX: <float=2>
		ExponentY: <float=0>
		Zx: <float=0>
		Zy: <float=0>
Location Params
	X: <float=0>; spinbox speed based on zoom
	Y: <float=0>; spinbox speed based on zoom
	Zoom: <float=0.25>; spinbox scaling logarythmic
	Log Zoom: <checkbox=false>
	Rotation: <float=0>
	SkewX: <float=0>
	SkewY: <float=0>
	Zoom Mode: <select>
		Options:
			"Y Stable"
			"X Stable"
			"Grow" (default)
			"Shrink"
Render Params
	Render Mode External: <select>
		Options:
			"Iteration Count" (default)
			"Exterior Distance Estimate"
	For "Iteration Count":
		Max Iteration Count: <int=10000>; range: x >= 0
		Iteration Count Scaling: <float=1>
		Normalized: <checkbox=false>
	Render Mode Internal: <select>
		Options:
			"Loop Iteration Count" (default)
			"Interior Distance Estimate"
	For "Loop Iteration Count":
		Max Loop Iteration Count: <int=10000>; range: x >= 0
		Loop Iteration Count Scaling: <float=1>
	Log View: <checkbox=false>
	Precision: <select>
		Options:
			"Float" (default)
			"Double"
			"Long Double" (if not available on processor then will say "Long Double (Emulated)" emulated using GMP or MPFR or MPIR)
			"Quad (GMP)" (possibly MPFR or MPIR instead)
			"Oct (GMP)" (possibly MPFR or MPIR instead)
			"BigDecimal (GMP)" (possibly MPIR instead)
Calculation Params
	Calculation: <select>
		Options:
			"Normal" (default)
			"Optimized Algo"
			"Vector Operations"
			"Shader"
	Threads: <int=CPU cores>; range: x >= 0
	Edge Tracing: <checkbox=false>
	Blitting: <select>
		Options:
			"FillRect (very slow)"
			"Bitmap + SetPixel (slow)"
			"Bitmap + PixelArray (medium)" (default)
			"Vulkan (fast)"
	Render to File: <button>
UI Params
	Dragging Mode: <select>
		Options:
			"Normal Mode" (default)
			"Centered Cursor"
	Live Update: <checkbox> (questionmark icon hover: Updates fractal for each character of input)
	Capture Volume Inputs: <checkbox> (questionmark icon hover: Converts volume up to rotate clockwise and down to counterclockwise, for keyboards with a volume knob)
	Reverse Volume Inputs: <checkbox> (greyed out unless capture volume inputs checked, but still editable despite greyed out)
Misc:
	Version: <info>
	Check for Updates: <button>
